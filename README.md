# 揭棋对战游戏

中国象棋揭棋变体，支持双人对战、人机对战、网络对战。

## 功能特性

- ✅ 双人对战 - 本地对战
- ✅ 人机对战 - 3档AI难度（简单/中等/困难）
- ✅ 网络对战 - 局域网对战
- ✅ 悔棋功能
- ✅ 走棋提示
- ✅ 排行榜

## 快速开始

### 方式1：直接运行（推荐）
双击 `启动游戏.bat`

### 方式2：命令行运行
```bash
python main.py
```

### 打包成exe
双击 `打包游戏.bat`，生成 `dist/揭棋对战.exe`

## 游戏规则

1. 棋子反扣随机放置，将帅为明棋
2. 未翻开的棋子按位置类型走法移动
3. 移动后棋子翻开，显示真实身份
4. 翻开后按标准象棋规则走棋
5. 士象可以过河（与标准象棋不同）
6. 将死或困毙对方帅将获胜

## 网络对战说明

1. 创建房间方执红，加入方执黑
2. 创建房间后，将显示的本机IP告诉对方
3. 对方输入IP点击"加入房间"即可开始对战

## 系统要求

- Python 3.6+
- 无需额外依赖（使用 Tkinter）
